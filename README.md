---

### Setup

```bash
git clone git@github.com:mhm13dev/vite-react-ts-yarn2-eslint-prettier-husky-gh-actions.git
```

### Install Yarn

```bash
npm install -g yarn
```

### Dependencies Installation

Change directory to project root and run:

```bash
yarn install
```

### Run Project

```bash
yarn dev
```

### Build Project

```bash
yarn build
```

### Lint Project

```bash
yarn lint
```

### Format Project

```bash
yarn format
```
